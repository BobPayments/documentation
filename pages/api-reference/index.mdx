Base URL: `https://api.payments.bob.company`

Todas as requisições precisam do header `Authorization: Bearer <sua-chave>`. Use `sk_test_` para sandbox e `sk_live_` para produção.

---

## Transações PIX

| Método | Endpoint | Descrição |
|--------|----------|-----------|
| `POST` | `/api/transactions` | Criar cobrança PIX |
| `GET` | `/api/transactions/:id` | Consultar status de uma transação |
| `GET` | `/api/transactions` | Listar transações do projeto |
| `POST` | `/api/sandbox/transactions/:id/pay` | Simular pagamento (sandbox) |

---

## Clientes

| Método | Endpoint | Descrição |
|--------|----------|-----------|
| `POST` | `/api/customers` | Criar ou atualizar cliente |
| `GET` | `/api/customers/:id` | Buscar cliente por ID |
| `GET` | `/api/customers` | Listar clientes do projeto |

## Loja

| Método | Endpoint | Descrição |
|--------|----------|-----------|
| `GET` | `/api/store` | Consultar configurações do projeto |
