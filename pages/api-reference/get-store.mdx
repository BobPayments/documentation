# Detalhes da Loja

Retorna informações do projeto (loja) vinculado à sua Secret Key.

## Autenticação

```
Authorization: Bearer sk_test_xxx
```

## Response (200)

```json
{
  "success": true,
  "data": {
    "id": "clx1234567890",
    "name": "Minha Loja",
    "devMode": true,
    "pixExpirationMinutes": 30,
    "maxTransactionAmountCents": null,
    "domains": [
      {
        "id": "cld1234567890",
        "host": "minhaloja.com"
      }
    ]
  }
}
```

<ResponseField name="data.devMode" type="boolean">
  `true` indica que o projeto está em modo desenvolvimento. Chaves `sk_live_*` são bloqueadas enquanto dev mode estiver ativo.
</ResponseField>
<ResponseField name="data.pixExpirationMinutes" type="integer">
  Tempo de expiração do PIX em minutos (padrão: 30).
</ResponseField>
<ResponseField name="data.maxTransactionAmountCents" type="integer | null">
  Valor máximo por transação em centavos. `null` = sem limite.
</ResponseField>
<ResponseField name="data.domains" type="array">
  Domínios registrados para o projeto. Apenas requisições com `Origin` desses domínios são aceitas em produção.
</ResponseField>

## Erros

| HTTP | Código | Descrição |
|------|--------|-----------|
| 401 | `ERR_AUTH_009` | Token inválido |
| 404 | `ERR_PROJECT_001` | Projeto não encontrado |
