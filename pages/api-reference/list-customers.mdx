# Listar Clientes

Retorna uma lista paginada de todos os clientes do projeto.

## Autenticação

```
Authorization: Bearer sk_test_xxx
```

## Query Parameters

<ParamField query="page" type="integer">
  Número da página. Default: `1`.
</ParamField>
<ParamField query="limit" type="integer">
  Itens por página (máximo 100). Default: `10`.
</ParamField>
<ParamField query="search" type="string">
  Texto de busca (máximo 200 caracteres). Pesquisa por nome, e-mail, CPF/CNPJ ou telefone do cliente.
</ParamField>
<ParamField query="dateFrom" type="string">
  Filtrar clientes criados a partir desta data (ISO 8601).
</ParamField>
<ParamField query="dateTo" type="string">
  Filtrar clientes criados até esta data (ISO 8601).
</ParamField>
