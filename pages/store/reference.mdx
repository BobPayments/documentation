## <Icon icon="folder-tree" type="solid" /> Estrutura

A loja representa o projeto associado à sua chave de API:

```json
{
  "id": "clz9x8w7v6u5t4s3r2q1",
  "name": "Minha Loja",
  "domains": ["loja.exemplo.com.br"],
  "webhookUrl": "https://loja.exemplo.com.br/webhooks/bob",
  "isSandbox": false,
  "createdAt": "2026-01-15T08:00:00.000Z"
}
```

## Atributos:

<AccordionGroup>
  <Accordion title="id:">
    ```json
    { "id": "clz9x8w7v6u5t4s3r2q1" }
    ```
    `id` : <u>string.</u><br />
    Identificador único do projeto no formato CUID.
  </Accordion>

  <Accordion title="name:">
    ```json
    { "name": "Minha Loja" }
    ```
    `name` : <u>string.</u><br />
    Nome do projeto configurado no Dashboard.
  </Accordion>

  <Accordion title="domains:">
    ```json
    { "domains": ["loja.exemplo.com.br"] }
    ```
    `domains` : <u>string[].</u><br />
    Lista de domínios autorizados para criar transações em produção. Obrigatório para ir a produção.
  </Accordion>

  <Accordion title="webhookUrl:">
    ```json
    { "webhookUrl": "https://loja.exemplo.com.br/webhooks/bob" }
    ```
    `webhookUrl` : <u>string | null.</u><br />
    URL de webhook configurada para receber notificações de eventos de pagamento.
  </Accordion>

  <Accordion title="isSandbox:">
    ```json
    { "isSandbox": true }
    ```
    `isSandbox` : <u>boolean.</u><br />
    Indica se a chave utilizada é de sandbox (`sk_test_*`) ou produção (`sk_live_*`).
  </Accordion>

  <Accordion title="createdAt:">
    ```json
    { "createdAt": "2026-01-15T08:00:00.000Z" }
    ```
    `createdAt` : <u>date-time.</u><br />
    Data e hora de criação do projeto.
  </Accordion>
</AccordionGroup>
