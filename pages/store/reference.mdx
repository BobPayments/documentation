## <Icon icon="folder-tree" type="solid" /> Estrutura

A loja representa o projeto associado à sua chave de API:

```json
{
  "id": "clz9x8w7v6u5t4s3r2q1",
  "name": "Minha Loja",
  "devMode": false,
  "pixExpirationMinutes": 60,
  "maxTransactionAmountCents": null
}
```

## Atributos:

<AccordionGroup>
  <Accordion title="id:">
    ```json
    { "id": "clz9x8w7v6u5t4s3r2q1" }
    ```
    `id` : <u>string.</u><br />
    Identificador único do projeto no formato CUID.
  </Accordion>

  <Accordion title="name:">
    ```json
    { "name": "Minha Loja" }
    ```
    `name` : <u>string.</u><br />
    Nome do projeto configurado no Dashboard.
  </Accordion>

  <Accordion title="devMode:">
    ```json
    { "devMode": false }
    ```
    `devMode` : <u>boolean.</u><br />
    Indica se o projeto está em modo sandbox (`true`) ou produção (`false`). Reflete o ambiente da chave de API utilizada (`sk_test_*` ou `sk_live_*`).
  </Accordion>

  <Accordion title="pixExpirationMinutes:">
    ```json
    { "pixExpirationMinutes": 60 }
    ```
    `pixExpirationMinutes` : <u>integer.</u><br />
    Tempo em minutos até uma cobrança PIX expirar. Configurável no Dashboard.
  </Accordion>

  <Accordion title="maxTransactionAmountCents:">
    ```json
    { "maxTransactionAmountCents": null }
    ```
    `maxTransactionAmountCents` : <u>integer | null.</u><br />
    Valor máximo permitido por transação, em centavos. `null` indica que não há limite configurado.
  </Accordion>

</AccordionGroup>
