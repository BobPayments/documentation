description: "Passos para ativar o ambiente de produção e começar a processar pagamentos reais."
icon: "rocket"
---

Quando sua integração estiver validada no sandbox, a transição para produção é simples.

<Steps>
  <Step title="Crie uma chave de produção">
    No dashboard → seu projeto → **Configurações → Chaves de API → Nova chave**. Selecione o ambiente **live** e associe à integração com os gateways de produção.
  </Step>
  <Step title="Desative o modo sandbox">
    O toggle fica no header do dashboard. Isso ativa as validações de produção.
  </Step>
  <Step title="Troque a chave de API">
    Substitua `sk_test_*` por `sk_live_*` no seu backend. Nenhuma outra mudança de código é necessária.
  </Step>
  <Step title="Valide com um pagamento real">
    Faça um teste com valor baixo (ex: R$ 3,50) e confirme que o webhook chegou corretamente.
  </Step>
</Steps>
