# Listar clientes

Retorna uma lista paginada de todos os clientes do seu projeto, ordenados por data de criação (mais recentes primeiro).

## Endpoint

```
GET /api/customers
```

## Parâmetros de query

<AccordionGroup>
  <Accordion title="page" icon="arrow-right">
    **number** · Opcional

    Número da página. Padrão: `1`.
  </Accordion>

  <Accordion title="limit" icon="list-ol">
    **number** · Opcional

    Quantidade de clientes por página. Padrão: `20`. Máximo: `100`.
  </Accordion>
</AccordionGroup>

## Exemplo de requisição

<CodeGroup>
```bash cURL
curl "https://api.payments.bob.company/api/customers?page=1&limit=20" \
  -H "Authorization: Bearer sk_test_sua_chave"
```

```javascript JavaScript
const params = new URLSearchParams({ page: '1', limit: '20' });

const response = await fetch(
  `https://api.payments.bob.company/api/customers?${params}`,
  {
    headers: { 'Authorization': 'Bearer sk_test_sua_chave' }
  }
);

const { data } = await response.json();
console.log(data.customers); // array de clientes
console.log(data.total);     // total de registros
```

```python Python
import requests

response = requests.get(
    'https://api.payments.bob.company/api/customers',
    headers={'Authorization': 'Bearer sk_test_sua_chave'},
    params={'page': 1, 'limit': 20}
)

result = response.json()['data']
print(result['customers'])
print(result['total'])
```
</CodeGroup>
