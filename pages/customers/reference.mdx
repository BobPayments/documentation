## <Icon icon="folder-tree" type="solid" /> Estrutura

Um cliente é representado em nossa API pela seguinte estrutura:

```json
{
  "id": "clx1a2b3c4d5e6f7g8h9",
  "name": "João Silva",
  "email": "joao@email.com",
  "phone": "11987654321",
  "document": "12345678900",
  "street": "Rua Exemplo",
  "streetNumber": "123",
  "complement": null,
  "neighborhood": "Centro",
  "zipCode": "01234567",
  "city": "São Paulo",
  "state": "SP",
  "country": "BR",
  "isBlocked": false,
  "blockedAt": null,
  "blockedReason": null,
  "blockExpiresAt": null,
  "blockType": null,
  "createdAt": "2026-02-28T09:00:00.000Z",
  "updatedAt": "2026-02-28T09:00:00.000Z"
}
```

## Atributos:

<AccordionGroup>
  <Accordion title="id:">
    ```json
    { "id": "clx1a2b3c4d5e6f7g8h9" }
    ```
    `id` : <u>string.</u><br />
    Identificador único do cliente no formato CUID.
  </Accordion>

  <Accordion title="name:">
    ```json
    { "name": "João Silva" }
    ```
    `name` : <u>string.</u><br />
    Nome completo do cliente.
  </Accordion>

  <Accordion title="email:">
    ```json
    { "email": "joao@email.com" }
    ```
    `email` : <u>string.</u><br />
    Endereço de e-mail do cliente.
  </Accordion>

  <Accordion title="phone:">
    ```json
    { "phone": "11987654321" }
    ```
    `phone` : <u>string.</u><br />
    Telefone com DDD, somente números.
  </Accordion>

  <Accordion title="document:">
    ```json
    { "document": "12345678900" }
    ```
    `document` : <u>string.</u><br />
    CPF ou CNPJ do cliente, somente números. Usado como identificador único por projeto.
  </Accordion>

  <Accordion title="street:">
    ```json
    { "street": "Rua Exemplo" }
    ```
    `street` : <u>string.</u><br />
    Nome da rua ou logradouro.
  </Accordion>

  <Accordion title="streetNumber:">
    ```json
    { "streetNumber": "123" }
    ```
    `streetNumber` : <u>string.</u><br />
    Número do endereço.
  </Accordion>

  <Accordion title="complement:">
    ```json
    { "complement": null }
    ```
    `complement` : <u>string | null.</u><br />
    Complemento do endereço (ex: `Apto 4`). Opcional.
  </Accordion>

  <Accordion title="neighborhood:">
    ```json
    { "neighborhood": "Centro" }
    ```
    `neighborhood` : <u>string.</u><br />
    Bairro do cliente.
  </Accordion>

  <Accordion title="zipCode:">
    ```json
    { "zipCode": "01234567" }
    ```
    `zipCode` : <u>string.</u><br />
    CEP somente números.
  </Accordion>

  <Accordion title="city:">
    ```json
    { "city": "São Paulo" }
    ```
    `city` : <u>string.</u><br />
    Cidade do cliente.
  </Accordion>

  <Accordion title="state:">
    ```json
    { "state": "SP" }
    ```
    `state` : <u>string.</u><br />
    Sigla do estado (UF).
  </Accordion>

  <Accordion title="country:">
    ```json
    { "country": "BR" }
    ```
    `country` : <u>string.</u><br />
    Código do país no formato ISO 3166-1 alpha-2.
  </Accordion>

  <Accordion title="isBlocked:">
    ```json
    { "isBlocked": false }
    ```
    `isBlocked` : <u>boolean.</u><br />
    Indica se o cliente está bloqueado. Clientes bloqueados não podem gerar novas transações.
  </Accordion>

  <Accordion title="blockedAt:">
    ```json
    { "blockedAt": "2026-02-28T10:00:00.000Z" }
    ```
    `blockedAt` : <u>date-time | null.</u><br />
    Data e hora em que o bloqueio foi aplicado.
  </Accordion>

  <Accordion title="blockedReason:">
    ```json
    { "blockedReason": "Suspeita de fraude" }
    ```
    `blockedReason` : <u>string | null.</u><br />
    Motivo registrado para o bloqueio do cliente.
  </Accordion>

  <Accordion title="blockExpiresAt:">
    ```json
    { "blockExpiresAt": null }
    ```
    `blockExpiresAt` : <u>date-time | null.</u><br />
    Data de expiração do bloqueio. `null` indica bloqueio permanente até desbloqueio manual.
  </Accordion>

  <Accordion title="blockType:">
    ```json
    { "blockType": null }
    ```
    `blockType` : <u>string | null.</u><br />
    Tipo de bloqueio aplicado.
  </Accordion>

  <Accordion title="createdAt:">
    ```json
    { "createdAt": "2026-02-28T09:00:00.000Z" }
    ```
    `createdAt` : <u>date-time.</u><br />
    Data e hora de criação do registro.
  </Accordion>

  <Accordion title="updatedAt:">
    ```json
    { "updatedAt": "2026-02-28T09:00:00.000Z" }
    ```
    `updatedAt` : <u>date-time.</u><br />
    Data e hora da última atualização.
  </Accordion>
</AccordionGroup>
