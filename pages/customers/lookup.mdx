# Buscar cliente por documento, email ou telefone

Localiza um cliente pelo CPF/CNPJ, endereço de e-mail ou número de telefone, sem precisar conhecer o ID interno.

## Endpoint

```
GET /api/customers/lookup
```

## Parâmetros de query

<AccordionGroup>
  <Accordion title="document" icon="id-card">
    **string** · Opcional

    CPF ou CNPJ do cliente, somente números.
  </Accordion>

  <Accordion title="email" icon="envelope">
    **string** · Opcional

    Endereço de e-mail do cliente.
  </Accordion>

  <Accordion title="phone" icon="phone">
    **string** · Opcional

    Telefone do cliente com DDD, somente números.
  </Accordion>
</AccordionGroup>

<Note>
  Informe pelo menos um dos parâmetros. Se mais de um for fornecido, a busca retorna clientes que correspondam a qualquer um deles.
</Note>

## Exemplo de requisição

<CodeGroup>
```bash cURL
# Buscar por CPF
curl "https://api.payments.bob.company/api/customers/lookup?document=12345678900" \
  -H "Authorization: Bearer sk_test_sua_chave"

# Buscar por e-mail
curl "https://api.payments.bob.company/api/customers/lookup?email=joao@email.com" \
  -H "Authorization: Bearer sk_test_sua_chave"
```

```javascript JavaScript
// Buscar por CPF
const response = await fetch(
  'https://api.payments.bob.company/api/customers/lookup?document=12345678900',
  {
    headers: { 'Authorization': 'Bearer sk_test_sua_chave' }
  }
);

const { data } = await response.json();

if (data) {
  console.log(data.id);   // ID do cliente encontrado
  console.log(data.name); // "João Silva"
} else {
  console.log('Cliente não encontrado');
}
```

```python Python
import requests

response = requests.get(
    'https://api.payments.bob.company/api/customers/lookup',
    headers={'Authorization': 'Bearer sk_test_sua_chave'},
    params={'email': 'joao@email.com'}
)

result = response.json()
data = result.get('data')

if data:
    print(data['id'])
    print(data['name'])
else:
    print('Cliente não encontrado')
```
</CodeGroup>

## Cliente não encontrado

Quando nenhum cliente corresponde à busca, a API retorna:

```json
{
  "success": true,
  "message": "Customer not found",
  "data": null
}
```

<Tip>
  Use este endpoint para verificar se um cliente já existe antes de criar uma nova transação, evitando duplicatas de cadastro.
</Tip>
