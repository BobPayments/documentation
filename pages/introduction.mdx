# Bem-vindo à Bob Payments

A Bob Payments é uma API de pagamentos PIX que permite criar cobranças, gerenciar clientes e receber notificações em tempo real via webhooks — com uma única integração.

## O que você pode fazer

<CardGroup cols={3}>
  <Card title="Cobranças PIX" icon="qrcode">
    Gere códigos PIX copia-e-cola em segundos e acompanhe o status em tempo real.
  </Card>
  <Card title="Clientes" icon="users">
    Cadastre e gerencie os compradores do seu projeto com histórico de transações.
  </Card>
  <Card title="Webhooks" icon="bell">
    Receba notificações automáticas quando um pagamento for confirmado ou expirar.
  </Card>
</CardGroup>

## Princípios da API

A API foi desenhada para ser previsível e fácil de integrar:

- **Endpoints intencionais** — cada rota faz exatamente o que o nome sugere
- **Autenticação simples** — Bearer Token em todos os endpoints
- **Respostas consistentes** — sempre `success`, `message` e `data`

```json
{
  "success": true,
  "message": "Transaction created successfully",
  "data": {
    "id": "clx7a8b9c0d1e2f3g4h5",
    "pixCode": "00020126580014br.gov.bcb.pix...",
    "expirationDate": "2026-02-28T10:00:00.000Z",
    "status": "waiting_payment"
  }
}
```
