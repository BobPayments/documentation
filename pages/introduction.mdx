Nesta documentação você encontrará tudo o que precisa para integrar com a API da Bob Payments. Desenvolvida por desenvolvedores para desenvolvedores, nossa plataforma foi projetada para ser simples, previsível e fácil de usar.

## <Icon icon="buildings" type="solid" /> O que é a Bob Payments?

A Bob Payments é um orquestrador de pagamentos PIX que surgiu da nossa necessidade de simplificar cobranças sem depender de um único gateway. Percebemos que integrar meios de pagamento era mais difícil do que precisava ser:

- Documentações extensas e confusas
- Múltiplos contratos e processos de homologação
- Instabilidade de um único gateway sem fallback
- Cada gateway com formato de API completamente diferente

Nossa solução? Uma API única que abstrai todos esses gateways e entrega o código copia-e-cola em uma chamada:

```javascript
// Criar uma cobrança PIX
const response = await fetch('https://api.payments.bob.company/api/v1/transactions/', {
  method: 'POST',
  headers: { 'Authorization': 'Bearer sk_test_sua_chave' },
  body: JSON.stringify({ amountCents: 10000, product: 'Plano Premium' })
});

const { data } = await response.json();
console.log(data.pixCode); // código copia-e-cola
```

## <Icon icon="laptop" type="solid" /> API descomplicada

Nossa API foi construída com três princípios fundamentais:

1. **Baseada em intenção** — cada endpoint representa exatamente o que você lê
   ```
   POST /api/v1/transactions/      // Cria uma cobrança PIX
   GET  /api/v1/transactions/{id}  // Busca uma transação específica
   GET  /api/v1/customers/         // Lista os clientes do projeto
   ```

2. **Idempotente** — use `externalId` para reenviar requisições com segurança, sem criar duplicatas

3. **Consistente** — todas as respostas seguem o mesmo formato:
   ```json
   {
     "success": true,
     "message": "Transaction created successfully",
     "data": {
       "id": "clx7a8b9c0d1e2f3g4h5",
       "status": "waiting_payment",
       "pixCode": "00020126580014br.gov.bcb.pix...",
       "expirationDate": "2026-02-28T10:00:00.000Z"
     }
   }
   ```
