# Atualizar Link de Pagamento

Atualiza parcialmente um link de pagamento. Todos os campos são opcionais — envie apenas os que deseja alterar.

## Path Parameters

<ParamField path="id" type="string" required>
  ID do link de pagamento.
</ParamField>

## Request Body

<ParamField body="domainIds" type="array">
  Substitui os domínios associados ao link. Mínimo 1 item.
</ParamField>

<ParamField body="layout" type="string">
  Layout do checkout. `CLASSIC` ou `MODERN`.
</ParamField>

<ParamField body="theme" type="string">
  Tema visual. `DEFAULT`, `DARK`, `MINIMAL` ou `VIBRANT`.
</ParamField>

<ParamField body="customAmountCents" type="integer">
  Novo valor em centavos (mínimo 100). Envie `null` para usar o preço do produto.
</ParamField>

<ParamField body="bannerUrl" type="string">
  URL do banner. Envie `null` para remover.
</ParamField>

<ParamField body="logoUrl" type="string">
  URL do logotipo. Envie `null` para remover.
</ParamField>

<ParamField body="primaryColor" type="string">
  Cor primária em hex `#RRGGBB`.
</ParamField>

<ParamField body="backgroundColor" type="string">
  Cor de fundo em hex `#RRGGBB`.
</ParamField>

<ParamField body="textColor" type="string">
  Cor do texto em hex `#RRGGBB`. Envie `null` para remover.
</ParamField>

<ParamField body="borderColor" type="string">
  Cor das bordas em hex `#RRGGBB`. Envie `null` para remover.
</ParamField>

<ParamField body="ctaText" type="string">
  Texto do botão (máx. 100 caracteres). Envie `null` para remover.
</ParamField>

<ParamField body="checkoutTitle" type="string">
  Título do checkout (máx. 200 caracteres). Envie `null` para remover.
</ParamField>

<ParamField body="topBannerText" type="string">
  Texto do banner superior (máx. 300 caracteres). Envie `null` para remover.
</ParamField>

<ParamField body="metaTitle" type="string">
  Título SEO (máx. 70 caracteres). Envie `null` para remover.
</ParamField>

<ParamField body="metaDescription" type="string">
  Descrição SEO (máx. 160 caracteres). Envie `null` para remover.
</ParamField>

<ParamField body="ogImageUrl" type="string">
  URL da imagem Open Graph. Envie `null` para remover.
</ParamField>

<ParamField body="active" type="boolean">
  Ativa (`true`) ou desativa (`false`) o link.
</ParamField>

<ParamField body="expiresAt" type="string">
  Nova data de expiração (ISO 8601). Envie `null` para remover a expiração.
</ParamField>

<ParamField body="successRedirectUrl" type="string">
  URL de redirecionamento pós-pagamento. Envie `null` para remover.
</ParamField>

<ParamField body="successMessage" type="string">
  Mensagem pós-pagamento (máx. 500 caracteres). Envie `null` para remover.
</ParamField>

<ParamField body="metadata" type="object">
  Objeto chave-valor livre. Envie `null` para limpar.
</ParamField>

## Erros

| HTTP | Código | Descrição |
|------|--------|-----------|
| 400 | `ERR_VALIDATION_001` | Body inválido |
| 401 | `ERR_AUTH_009` | Token inválido |
| 404 | — | Link de pagamento não encontrado |
| 500 | — | Erro interno do servidor |
