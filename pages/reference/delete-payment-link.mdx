# Deletar Link de Pagamento

Remove um link de pagamento. Esta ação é irreversível — a URL do checkout deixará de funcionar imediatamente.

<Warning>
  Prefira **desativar** o link (`active: false` via `PUT`) em vez de deletar, para preservar o histórico de visualizações.
</Warning>

## Path Parameters

<ParamField path="id" type="string" required>
  ID do link de pagamento a ser deletado.
</ParamField>

## Response (204)

Sem corpo de resposta.

## Erros

| HTTP | Código | Descrição |
|------|--------|-----------|
| 401 | `ERR_AUTH_009` | Token inválido |
| 404 | — | Link de pagamento não encontrado |
| 500 | — | Erro interno do servidor |
