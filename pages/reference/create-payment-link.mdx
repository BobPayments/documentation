# Criar Link de Pagamento

Cria uma nova página de checkout vinculada a um produto e um ou mais domínios.

## Request Body

<ParamField body="integrationId" type="string" required>
  ID da integração (gateway) que processará os pagamentos deste link.
</ParamField>

<ParamField body="productId" type="string" required>
  ID do produto que será exibido no checkout.
</ParamField>

<ParamField body="domainIds" type="array" required>
  Lista de IDs de domínios onde o link estará disponível. Mínimo 1.
</ParamField>

<ParamField body="layout" type="string">
  Layout da página de checkout. `CLASSIC` ou `MODERN`. Padrão: `CLASSIC`.
</ParamField>

<ParamField body="theme" type="string">
  Tema visual. `DEFAULT`, `DARK`, `MINIMAL` ou `VIBRANT`. Padrão: `DEFAULT`.
</ParamField>

<ParamField body="customAmountCents" type="integer">
  Valor customizado em centavos (mínimo 100). Se omitido, usa o preço do produto.
</ParamField>

<ParamField body="tagIds" type="array">
  IDs de tags para organização interna.
</ParamField>

<ParamField body="bannerUrl" type="string">
  URL da imagem de banner exibida no topo do checkout.
</ParamField>

<ParamField body="logoUrl" type="string">
  URL do logotipo exibido no checkout.
</ParamField>

<ParamField body="primaryColor" type="string">
  Cor primária em hex `#RRGGBB`. Padrão: `#6366f1`.
</ParamField>

<ParamField body="backgroundColor" type="string">
  Cor de fundo em hex `#RRGGBB`. Padrão: `#ffffff`.
</ParamField>

<ParamField body="textColor" type="string">
  Cor do texto em hex `#RRGGBB`. Opcional.
</ParamField>

<ParamField body="borderColor" type="string">
  Cor das bordas em hex `#RRGGBB`. Opcional.
</ParamField>

<ParamField body="ctaText" type="string">
  Texto do botão de ação (máx. 100 caracteres). Ex: `"Pagar agora"`.
</ParamField>

<ParamField body="checkoutTitle" type="string">
  Título exibido no topo do checkout (máx. 200 caracteres).
</ParamField>

<ParamField body="topBannerText" type="string">
  Texto do banner superior (máx. 300 caracteres). Ex: `"Frete grátis hoje!"`.
</ParamField>

<ParamField body="metaTitle" type="string">
  Título SEO da página (máx. 70 caracteres).
</ParamField>

<ParamField body="metaDescription" type="string">
  Descrição SEO da página (máx. 160 caracteres).
</ParamField>

<ParamField body="ogImageUrl" type="string">
  URL da imagem para compartilhamento em redes sociais (Open Graph).
</ParamField>

<ParamField body="expiresAt" type="string">
  Data/hora de expiração do link (ISO 8601). Após essa data o checkout exibirá erro.
</ParamField>

<ParamField body="successRedirectUrl" type="string">
  URL para redirecionar o cliente após o pagamento confirmado.
</ParamField>

<ParamField body="successMessage" type="string">
  Mensagem exibida ao cliente após o pagamento (máx. 500 caracteres).
</ParamField>

<ParamField body="orderBumps" type="array">
  Produtos adicionais exibidos no checkout. Máximo 5.
  <Expandable title="Campos de cada order bump">
    <ParamField body="productId" type="string" required>
      ID do produto do order bump.
    </ParamField>
    <ParamField body="customAmountCents" type="integer">
      Valor customizado em centavos (mínimo 100).
    </ParamField>
    <ParamField body="heading" type="string">
      Título do order bump (máx. 200 caracteres).
    </ParamField>
    <ParamField body="bumpDescription" type="string">
      Descrição do order bump (máx. 500 caracteres).
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="metadata" type="object">
  Objeto chave-valor livre para armazenar dados internos.
</ParamField>

<ResponseField name="data.id" type="string">
  ID único do link de pagamento.
</ResponseField>
<ResponseField name="data.shortCode" type="string">
  Código curto usado na URL do checkout. Ex: `https://seu-dominio.com/abc123`.
</ResponseField>
<ResponseField name="data.domains" type="array">
  Domínios associados com a URL completa de cada um.
</ResponseField>
<ResponseField name="data.active" type="boolean">
  Se o link está ativo e aceitando pagamentos.
</ResponseField>

## Erros

| HTTP | Código | Descrição |
|------|--------|-----------|
| 400 | `ERR_VALIDATION_001` | Body inválido |
| 401 | `ERR_AUTH_009` | Token inválido |
| 404 | — | Produto, integração ou domínio não encontrado |
| 409 | — | Conflito (shortCode duplicado) |
| 422 | — | Entidade não processável |
| 500 | — | Erro interno do servidor |
