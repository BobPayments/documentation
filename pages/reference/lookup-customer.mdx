# Buscar Cliente por Chave

Retorna o perfil completo de um cliente usando telefone, CPF/CNPJ ou e-mail como chave de busca. Informe exatamente **um** dos parâmetros disponíveis.

## Query Parameters

<ParamField query="phone" type="string">
  Número de telefone do cliente (máximo 20 caracteres).
</ParamField>
<ParamField query="document" type="string">
  CPF ou CNPJ do cliente (máximo 20 caracteres).
</ParamField>
<ParamField query="email" type="string">
  E-mail do cliente (máximo 200 caracteres).
</ParamField>

<Note>
  Informe **exatamente um** dos parâmetros acima. Enviar mais de um ou nenhum retorna erro 400.
</Note>

## Erros

| HTTP | Código | Descrição |
|------|--------|-----------|
| 400 | `ERR_VALIDATION` | Nenhum ou mais de um parâmetro informado |
| 401 | `ERR_AUTH_009` | Token inválido |
