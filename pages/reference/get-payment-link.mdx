# Buscar Link de Pagamento

Retorna todas as informações de um link de pagamento específico, incluindo produto, order bumps, branding, SEO e configurações de conversão.

## Path Parameters

<ParamField path="id" type="string" required>
  ID do link de pagamento.
</ParamField>

<ResponseField name="paymentLink.shortCode" type="string">
  Código curto que compõe a URL do checkout.
</ResponseField>
<ResponseField name="paymentLink.branding" type="object">
  Configurações visuais do checkout (cores, logo, banner).
</ResponseField>
<ResponseField name="paymentLink.content" type="object">
  Textos exibidos no checkout (CTA, título, banner superior).
</ResponseField>
<ResponseField name="paymentLink.seo" type="object">
  Metadados SEO e Open Graph da página.
</ResponseField>
<ResponseField name="paymentLink.viewCount" type="integer">
  Total de acessos à página de checkout.
</ResponseField>

## Erros

| HTTP | Código | Descrição |
|------|--------|-----------|
| 401 | `ERR_AUTH_009` | Token inválido |
| 404 | — | Link de pagamento não encontrado |
| 500 | — | Erro interno do servidor |
