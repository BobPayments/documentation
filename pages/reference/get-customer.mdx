# Buscar Cliente

Retorna os detalhes completos de um cliente, incluindo endereço, status de bloqueio e as transações mais recentes.

## Path Parameters

<ParamField path="id" type="string" required>
  ID do cliente (formato CUID).
</ParamField>


<ResponseField name="data.isBlocked" type="boolean">
  Indica se o cliente está bloqueado. Clientes bloqueados não podem criar novas transações.
</ResponseField>
<ResponseField name="data.blockedAt" type="string | null">
  Data/hora do bloqueio (ISO 8601). `null` se não está bloqueado.
</ResponseField>
<ResponseField name="data.blockedReason" type="string | null">
  Motivo do bloqueio. `null` se não está bloqueado.
</ResponseField>
<ResponseField name="data.blockExpiresAt" type="string | null">
  Data/hora de expiração do bloqueio (ISO 8601). `null` se o bloqueio é permanente ou não está bloqueado.
</ResponseField>
<ResponseField name="data.blockType" type="string | null">
  Tipo do bloqueio (ex: `manual`, `fraud`). `null` se não está bloqueado.
</ResponseField>
<ResponseField name="data.recentTransactions" type="array">
  Últimas transações do cliente com id, valor, status, produto e datas.
</ResponseField>
<ResponseField name="data.totalTransactions" type="integer">
  Número total de transações do cliente no projeto.
</ResponseField>

## Erros

| HTTP | Código | Descrição |
|------|--------|-----------|
| 401 | `ERR_AUTH_009` | Token inválido |
| 404 | `ERR_CUSTOMER_001` | Cliente não encontrado |
