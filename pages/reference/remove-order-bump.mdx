# Remover Order Bump

Remove um produto da lista de order bumps do checkout.

## Path Parameters

<ParamField path="id" type="string" required>
  ID do link de pagamento.
</ParamField>

<ParamField path="productId" type="string" required>
  ID do produto a ser removido dos order bumps.
</ParamField>

## Response (204)

Sem corpo de resposta.

## Erros

| HTTP | Código | Descrição |
|------|--------|-----------|
| 401 | `ERR_AUTH_009` | Token inválido |
| 404 | — | Link ou order bump não encontrado |
| 500 | — | Erro interno do servidor |
